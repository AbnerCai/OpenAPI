# 删除文摘API

**简要描述：** 

- 删除文摘接口

**请求URL：** 
- ` http://ip:port/v1/digests/:id?token=token `
  
**请求方式：**
- DELETE 

**参数：** 

> 参照文摘数据字典

 **请求示例**

` http://ip:port/v1/digests/5cbf2ec59d26a01a48049ebd?token=ad50c4a05b4949498f875c488e04f243 `

 **返回参数说明** 

> 参照文摘数据字典

 **返回示例**
```json
{
    "code": 0,
    "msg": "删除成功！"
}
```
