# 新建文摘API

**简要描述：** 

- 新建文摘接口

**请求URL：** 
- ` http://ip:port/v1/digests `
  
**请求方式：**
- POST 

**参数：** 

> 参照文摘数据字典

 **请求示例**

```json
{
	"token": "83f9e624dc4148ecb735f4be4e9553fe",
	"content": "生之有崖，学而无涯。"
}
```

 **返回参数说明** 

> 参照文摘数据字典

 **返回示例**
```json
{
    "code": 0,
    "msg": "创建成功！",
    "data": {
        "content": "生之有崖，学而无涯。",
        "createTime": "2019-04-23T15:11:42.754Z"
    }
}
```
